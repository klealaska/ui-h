## How to implement

Import the following packages `import { MatSnackBar, MatSnackBarRef } from '@angular/material/snack-bar';`.

Define snackbarRef and set the MatSnackBarRef type to <code>ToastComponent</code>. Declare snackBar inside your constructor. Additionally, Type aliases for horizontalPosition and verticalPosition can be found [here](https://material.angular.io/components/snack-bar/api#MatSnackBarHorizontalPosition).
