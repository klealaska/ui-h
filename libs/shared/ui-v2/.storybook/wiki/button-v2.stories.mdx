# Inputs

## Imports

Component - `import { ButtonComponent, AnchorComponent } from '@ui-coe/shared/ui-v2';`

Enums - `import { ButtonTypes, ButtonSizes, ButtonColors, ButtonHTMLTypes } from '@ui-coe/shared/ui-v2';`

## Description

Input can me implemented by using `ax-button` or `ax-anchor` if you need an `href`

These utilized the `ButtonStyles`, `ButtonColors`, `ButtonSizes`, `ButtonHTMLTypes` enums

No properties are needed to be passed in to get a basic button

```
<ax-button>Button</ax-button>
```

Change the style by passing in `ButtonStyles` enum. Supports `primary`, `secondary`, or `tertiary`

```
<ax-button [style]="ButtonStyles.primary">Button</ax-button>
```

Change the color by using the `ButtonColors` enum. Supports `default`, `critical`, or `neutral`

```
<ax-button [color]="ButtonColors.default">Button</ax-button>
```

Change size by passing in `ButtonSizes` enum. Supports `sm`, `md`, or `lg`

```
<ax-button [size]="ButtonSizes.lg">Button</ax-button>
```

Buttons support passing `mat-icons` into them. Pass them in by themselves or to the left or right of some text

```
<ax-button>
  <mat-icon>add<mat-icon>
<ax-button>
```

```
<ax-button>
  <mat-icon>add<mat-icon>Button
<ax-button>
```

```
<ax-button>
  Button<mat-icon>add<mat-icon>
<ax-button>
```
