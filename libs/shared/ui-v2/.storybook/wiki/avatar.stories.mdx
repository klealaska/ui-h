## Avatar

**A working example of this Avatar implementation can be found @ \_libs/shared/ui-v2/src/lib/avatar/avatar.component.stories.ts **

Avatar can be implemented by using the `ax-avatar` component. Example provided below.

```
<ax-avatar
  size="md"
  img="https://mdbcdn.b-cdn.net/img/new/avatars/8.webp"
  [name]="name"
></ax-avatar>

```
