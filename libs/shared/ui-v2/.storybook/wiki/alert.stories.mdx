## Alert

**A working example of this Alert implementation can be found @ \_libs/shared/ui-v2/src/lib/alert/alert.component.stories.ts **

Alert can be implemented by using the `ax-alert` component. Example provided below.

```
   <ax-alert
      [type]="type"
      [title]="title"
      [message]="message"
      [closable]="closable"
      [showIcon]="showIcon"
      [action]="actionTemplate"
      (onhide)="onhide($event)"
    >
      <ng-template #actionTemplate>
        <button mat-flat-button color="primary" class="md" (click)="actionClick()">
        Button
      </button>
      </ng-template>
    </ax-alert>

```

Button needs to implement by using ng-template, example provied below.

```
<ng-template #actionTemplate>
  <button mat-flat-button color="primary" class="md" (click)="actionClick()">
    Button
  </button>
</ng-template>

```

  </tr>
  <tr>
    <td>@Input() closable: boolean</td>

  </tr>
