## Header

**A working example of this Header implementation can be found @ \_libs/shared/ui-v2/src/lib/header/header.component.stories.ts **

Header can be implemented by using the `ax-header` component. Example provided below.

```
<ax-header
  logoImgSrc = ''
  [avatarInput] = 'avatarInput'
></ax-header>

```
