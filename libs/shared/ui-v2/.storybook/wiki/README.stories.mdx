# **Overview**

**This README docs includes any and all information needed to consume and begin working with the components in the ui-v2 library as well as the styles in dsm-styles-v2 library**

To inherit styles from the dsm-styles-v2 lib you will need to initilize the dsm-styles-v2 lib in your apps project.json file.
View the reference image below to see an example of this.

You will also need to import these three tailwind configs inside your apps root <code>styles.scss</code> stylesheet. <code>@tailwind base; @tailwind components; @tailwind utilities;</code>
