## Sidenav

**A working example of this Side implementation can be found @ \_libs/shared/ui-v2/src/lib/header/header.component.stories.ts **

Sidenav can be implemented by using the `ax-sidenav` component. Example provided below.

```
<ax-sidenav
  [navItems]='navItems'
></ax-sidenav>

```
