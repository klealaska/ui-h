## Tag

Tag can be implemented by using the `ax-tag` component

## Directives

<table>
  <tr>
    <th>Name</th>
    <th>Args</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>@Input() size: TagSize</td>
    <td>'sm' | 'md' | 'lg'</td>
    <td>Sets the tag size.</td>
  </tr>
  <tr>
    <td>@Input() type: TagType</td>
    <td>'default' | 'success' | 'warning' | 'critical' | 'informational'</td>
    <td>Sets the tag type.</td>
  </tr>
  <tr>
    <td>@Input() style: TagStyle</td>
    <td>'filled' | 'border'</td>
    <td>Sets the tag style.</td>
  </tr>
  <tr>
    <td>@Input() icon: string</td>
    <td>''</td>
    <td>Sets icon to the right of tag text.</td>
  </tr>
  <tr>
    <td>@Input() text: string</td>
    <td>''</td>
    <td>Sets the text inside of tag.</td>
  </tr>
</table>
