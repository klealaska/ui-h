Html - `<div class="bg-primary-800"></div>`

Css - `background theme('colors.primary.800')`

# **Additional Specifications**

## Accessing the Colors

To access the color palette you will need to have Tailwind enabled in your app.

If you generate a new app, Tailwind will be added automatically. If you're an existing app without Tailwind, use the generator `@nx/angular:setup-tailwind` and replace the config with the code below.

```
const { createGlobPatternsForDependencies } = require('@nx/angular/tailwind');
const { join } = require('path');
const sharedTailwindConfig = require('../../tailwind.config');

module.exports = {
  presets: [sharedTailwindConfig],

  content: [
    join(__dirname, 'src/**/!(*.stories|*.spec).{ts,html}'),

    ...createGlobPatternsForDependencies(__dirname),
  ],
};

```
