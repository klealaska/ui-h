## Inputs

Input can me implemented by using the `ax-input` component

## Directives

<table>
  <tr>
    <th>Name</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>@Input() fixed: boolean</td>
    <td>Sets a fixed width. Default is false</td>
  </tr>
  <tr>
    <td>@Input() disabled: boolean</td>
    <td>Sets the disabled state</td>
  </tr>
  <tr>
    <td>@Input() readOnly: boolean</td>
    <td>Sets the readOnly state</td>
  </tr>
  <tr>
    <td>@Input() id: string</td>
    <td>Adds an id to the input. Used for accessible (required if using a label)</td>
  </tr>
  <tr>
    <td>@Input() label: string</td>
    <td>Title for an input</td>
  </tr>
  <tr>
    <td> @Input() tooltip: InputTooltip</td>
    <td>
      Adds a tooltip beside label. Only need to pass in tooltipText. Other properties can be over
      written
    </td>
  </tr>
  <tr>
    <td> @Input() maxLength: number</td>
    <td>Sets the max length. Default 50</td>
  </tr>
  <tr>
    <td>@Input() inputType: InputType</td>
    <td>Input type of the element. Default is text</td>
  </tr>
  <tr>
    <td>@Input() placeholder: string</td>
    <td>Placeholder text</td>
  </tr>
  <tr>
    <td>@Input() control: UntypedFormControl</td>
    <td>Adds a formControl to an input</td>
  </tr>
  <tr>
    <td>@Input() leftIcon: string</td>
    <td>Icon inside input to the left</td>
  </tr>
  <tr>
    <td>@Input() leftIconButton: boolean</td>
    <td>Makes leftIcon into a clickable icon</td>
  </tr>
  <tr>
    <td>@Input() rightIcon: string</td>
    <td>Icon inside input to the right</td>
  </tr>
  <tr>
    <td>Input() rightIconButton: boolean</td>
    <td>Makes rightIcon into a clickable icon</td>
  </tr>
  <tr>
    <td>@Input() buttonIcon: string</td>
    <td>Clickable icon inside the input</td>
  </tr>
  <tr>
    <td>@Input() hintMessage: string</td>
    <td>Hint text</td>
  </tr>
  <tr>
    <td>@Input() error: FormFieldError</td>
    <td>Object that takes in message and optional icon to populate an error message</td>
  </tr>
  <tr>
    <td>@Output() leftIconButtonEvent = new EventEmitter()</td>
    <td>Emits that an icon button has been clicked</td>
  </tr>
  <tr>
    <td>@Output() rightIconButtonEvent = new EventEmitter()</td>
    <td>Emits that an icon button has been clicked</td>
  </tr>
  <tr>
    <td>@Output() inputValueEvent = new EventEmitter()</td>
    <td>Emits the value of an input</td>
  </tr>
</table>
