## Checkbox

**A working example of this Checkbox implementation can be found @ \_libs/shared/ui-v2/src/lib/Checkbox/Checkbox.component.stories.ts **

The checkbox component can be implemented in two ways

**1.Individual checkbox without layout and heading**

Individual checkbox can be implemented by using the `ax-checkbox` component. Example provided below.

```
<ax-checkbox
        [labelPosition]="labelPosition"
        (checkedChange)="checkedChange($event)"
        [(ngModel)]="selectedValue"
        name="checkbox1"
        value="optionA"
        [label]="label"
        [disabled]="disabled"
        [indeterminate]="true"
        [noText]="false"
        [fixed]="true"
  ></ax-checkbox>

```

**2.Group of checkbox with layout and heading**

Checkbox Group can be implemented by using the `ax-checkbox-group` component. Example provided below.

```

<ax-checkbox-group
        [layout]="layout"
        [labelHeading]="labelHeading"
        [disabled]="disabled"
        name="checkboxButton2"
        (checkedChange)="checkedChange($event)"
>
    <ax-checkbox [labelPosition]="labelPosition" [(ngModel)]="selectedValue1" value="optionA" [fixed]="true">{{ label }}</ax-checkbox>
    <ax-checkbox [labelPosition]="labelPosition" [(ngModel)]="selectedValue2" value="optionB" [fixed]="true">{{ label }}</ax-checkbox>
</ax-checkbox-group>

```
