# Split Button

## Imports

Component - `import { SplitButtonComponent } from '@ui-coe/shared/ui-v2';`

Enums - `import { ButtonTypes, ButtonSizes, ButtonColors } from '@ui-coe/shared/ui-v2';`

## Description

Split button can be implemented by using `ax-split-button`.

These utilized the `ButtonStyles`, `ButtonColors`, `ButtonSizes`, enums

No properties are needed to be passed in to get a basic button. However you do need to provide dropdown array property.

```
<ax-split-button [dropdown]="dropdown">Button</ax-split-button>
```

Change the style by passing in `ButtonStyles` enum. Supports `primary`, `secondary`, or `tertiary`

```
<ax-split-button [style]="ButtonStyles.primary">Button</ax-split-button>
```

Change the color by using the `ButtonColors` enum. Supports `default`, `critical`, or `neutral`

```
<ax-split-button [color]="ButtonColors.default">Button</ax-split-button>
```

Change size by passing in `ButtonSizes` enum. Supports `sm`, `md`, or `lg`

```
<ax-split-button [size]="ButtonSizes.lg">Button</ax-split-button>
```
