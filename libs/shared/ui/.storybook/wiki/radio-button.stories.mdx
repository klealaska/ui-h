# **Overview**

`<mat-radio-button>` provides the same functionality as a native `<input type="radio">` enhanced with Material Design styling and animations.

<img src={RadioButton} alt="RadioButton" />

# **Additional Specifications**

- Needs `color` property set to `primary`

# **Additional Accessibility Requirements**

None

# **Angular Material**

Review Angular Material for current documentation on how to implement the Radio button component.

https://material.angular.io/components/radio/overview
