# **Overview**

`<mat-spinner>` is a circular indicator of progress and activity.

`<mat-progress-spinner>` supports two modes: **determinate** and **indeterminate**.

Determinate mode will require a value property in order for it to work.

<img src={Spinner} alt="Spinner" />

# **Additional Specifications**

None

# **Additional Accessibility Requirements**

- Each progress spinner should be given a meaningful label via `aria-label` or `aria-labelledby`
