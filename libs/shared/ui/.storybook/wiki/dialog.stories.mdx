# **Overview**

The Document Dialog component allows for styling of a dialog with or without a call to action (CTA).

<img src={Dialog} alt="Dialog" />

# **Additional Specifications**

## Building the Document Dialog

```
export interface DialogData {

  title?: string;

  message: string;

  icon?: string;

  cta: boolean;

  initAction: string;

}
```

_The_ `title` _and_ `icon` _are optional_.

## Call to Action

The CTA uses Boolean strategy that enables dynamic buttons based on whether you pass `true` or `false`. If the CTA is true, pass the information you want inside the button. If the CTA is false, a **Dismiss** button will display.
