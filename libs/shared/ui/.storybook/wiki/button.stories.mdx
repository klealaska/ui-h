Support Basic, Flat, Stroked, and Icon. All of these buttons use `color="primary"` and are given a `sm`, `md`, or `lg` class depending on the size.

<br></br>
<img src={FlatBtn} alt="Flat Button" width="140" />

```
<button mat-flat-button color="primary" class="lg">Primary</button>
```

```
<button mat-flat-button mat-icon-button color="primary" class="lg"> <mat-icon class="lg">add</mat-icon> </button>
```
