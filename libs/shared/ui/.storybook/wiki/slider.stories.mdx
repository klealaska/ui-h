# **Overview**

`<mat-slider>` allows for the selection of a value from a range via mouse, touch, or keyboard, similar to `<input type="range">`.

<img src={Slider} alt="Slider" />

# **Additional Specifications**

- Color property must be set to `"primary"`

# **Additional Accessibility Requirements**

None

# **Angular Material**

Review Angular Material for current documentation on how to implement the Slider component.
