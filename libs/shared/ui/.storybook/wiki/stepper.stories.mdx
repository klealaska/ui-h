# **Overview**

The Document Stepper allows users to indicate progress in defined stages. Both a horizontal and a vertical stepper have been designed to provide options to display progress.

# **Additional Specifications**

## Stepper Color

- Needs `color` property set to `accent`

`<mat-horizontal-stepper color="accent">` -or- `<mat-vertical-stepper color="accent">`

## Stepper Connecting Lines

To ensure the lines between the icons change, add a variable declaration and custom class to the selected index.

`<mat-horizontal-stepper #horizontalStepper [class]="'selectedIndex' + horizontalStepper.selectedIndex">`

-or-

`<mat-vertical-stepper #verticalStepper [class]="'selectedIndex' + verticalStepper.selectedIndex">`

## Stepper Sizes

Steppers in three sizes are available for `<mat-horizontal-stepper>` and `<mat-vertical-stepper>`.

Small

- The default material Stepper

Medium

- Requires `class='mat-stepper-medium'`

Large

- Requires `class='mat-stepper-large'`
