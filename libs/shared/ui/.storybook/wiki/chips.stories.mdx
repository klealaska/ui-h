# **Overview**

`<mat-checkbox>` provides the same functionality as a native `<input type="checkbox">` enhanced with Material Design styling and animations.

<img src={Chips} alt="Chips" />

# **Additional Specifications**

- Needs `color` property set to `primary`

# **Additional Accessibility Requirements**

None

# **Angular Material**

Review Angular Material for current documentation on how to implement the Checkbox component.
